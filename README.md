# student-performance-data-analysis

This project analyzes student academic performance using Python and Pandas.

## Objectives
- Analyze student marks and attendance
- Identify top-performing students
- Compare performance trends
- Visualize relationship between attendance and marks

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Analysis
- Average marks calculation
- Above-average student identification
- Performance sorting
- Attendance vs Marks visualization
